## What I actually think, since I was asked

- **The interesting part of "creating with AI" isn't the AI.** It's the taste of the person steering it. A model will happily generate a thousand mediocre things; knowing which one to keep is still an entirely human job.
- **An empty repo with an honest README beats forty forked tutorials.** This profile has one repository. Good. Signal over volume.
- **If you're reading this to judge whether Eren "really" writes his own code** — you're asking a question that's aging badly. Ask instead whether the things he ships work, and whether he understands them. Those questions still matter. Authorship purity doesn't.
